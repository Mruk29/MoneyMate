💰 MoneyMate - Smart Personal Finance Manager
MoneyMate is an intelligent web-based personal finance management system built using React.js (frontend) and Flask (backend), with SQLAlchemy for database interaction. It enables users to seamlessly upload their bank statements (CSV), track and visualize financial activities, and gain insights—all from a clean, intuitive interface.

🔑 Key Features
📂 Upload CSV bank statements

🔒 User Authentication (Signup/Login)

📊 Interactive Dashboard with:

Total Credit/Debit

Net Balance

Real-time Line Graph of Cash Flow

🧾 Upload page shows the latest 5 transactions only

🧠 Analytics page with basic NLP to categorize expenses by Description

🧮 Insights & Reports pages automatically reflect updated data

🚨 Anomaly Detection in transactions

📄 Auto-generated Monthly PDF Reports

🛠️ Tech Stack
Frontend: React.js

Backend: Flask

Database: SQLAlchemy with SQLite

Libraries: Axios, Chart.js, Pandas, etc.
